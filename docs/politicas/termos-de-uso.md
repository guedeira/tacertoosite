# Termos de Uso

Última atualização: 24 de abril de 2026

## 1. Objetivo da aplicação

O **Tá certo o site?** é uma ferramenta informativa que compara uma URL ou domínio informado pelo usuário com domínios oficiais previamente cadastrados para uma marca.

A aplicação tem como objetivo ajudar o usuário a perceber se o domínio informado corresponde ou não ao domínio oficial cadastrado na nossa base.

## 2. O que a aplicação faz

A aplicação:

- recebe uma URL ou domínio informado pelo usuário;
- extrai e normaliza o domínio;
- compara esse domínio com os domínios oficiais cadastrados para a marca selecionada;
- informa se houve correspondência com a base cadastrada.

## 3. O que a aplicação não faz

A aplicação **não garante que um site é seguro**.

A aplicação também não realiza:

- análise de vírus, malware ou phishing;
- consulta a listas de bloqueio;
- verificação de certificado digital;
- análise de reputação;
- consulta de DNS;
- auditoria do conteúdo da página;
- validação jurídica, financeira ou técnica sobre a empresa.

Um domínio corresponder ao cadastro não significa, por si só, que toda ação realizada naquele site é segura ou recomendada.

## 4. Responsabilidade do usuário

O usuário é o único responsável por quaisquer decisões tomadas a partir das informações exibidas pela aplicação.

Antes de informar dados pessoais, senhas, códigos, informações bancárias ou realizar pagamentos, o usuário deve conferir a informação por canais oficiais da marca ou instituição envolvida.

A aplicação serve apenas como apoio informativo e não substitui cautela, análise própria ou orientação especializada.

## 5. Base de domínios

Os domínios oficiais são cadastrados manualmente e podem estar incompletos, desatualizados ou sujeitos a erro.

Solicitações de inclusão, correção ou remoção de empresas e domínios podem ser enviadas para revisão. O envio de uma solicitação não garante aprovação ou inclusão na base.

## 6. Limitação de responsabilidade

Na máxima extensão permitida pela legislação aplicável, os responsáveis pela aplicação não se responsabilizam por perdas, danos, prejuízos, fraudes, acessos indevidos, decisões financeiras, exposição de dados ou quaisquer consequências decorrentes do uso das informações apresentadas.

## 7. Uso adequado

O usuário se compromete a usar a aplicação de forma legítima, sem tentar prejudicar, sobrecarregar, explorar falhas ou interferir no funcionamento do serviço.
